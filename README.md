# Лендинг «ИИ-мультики» · Курс Умиды

Одностраничный сайт курса по созданию ИИ-роликов. Чистый HTML/CSS/JS — без сборки и зависимостей.

## Структура проекта

```
project/
├─ index.html            ← разметка страницы
├─ css/
│  └─ styles.css         ← все стили
├─ js/
│  └─ script.js          ← аккордеоны, меню, анимации появления
├─ assets/
│  ├─ favicon.svg        ← иконка вкладки (клубничка)
│  └─ img/
│     ├─ hero.jpg            ← фото в первом экране (Умида + персонаж)
│     ├─ about.jpg           ← фото в блоке «Об авторе» (с планшетом)
│     ├─ character-1.jpg     ← галерея: Неидеальный наследник
│     ├─ character-2.jpg     ← галерея: Неидеальный наследник
│     ├─ character-3.jpg     ← галерея: Подлая подружка
│     ├─ character-4.jpg     ← галерея: Драгоценная месть
│     └─ character-5.jpg     ← галерея: Авторский персонаж
└─ README.md
```

## Как открыть в VS Code

1. Откройте папку `project` в VS Code (File → Open Folder).
2. Установите расширение **Live Server** (автор: Ritwick Dey).
3. Правый клик по `index.html` → **Open with Live Server**.
4. Сайт откроется в браузере и будет обновляться при сохранении файлов.

Можно и без Live Server — просто откройте `index.html` двойным кликом в браузере.

## Как заменить фото

Положите новые изображения в `assets/img/` с теми же именами (`hero.jpg`, `about.jpg`, `character-1.jpg` … `character-5.jpg`) — менять код не нужно.
Рекомендуемая ширина: hero ≈ 760px, about ≈ 680px, персонажи ≈ 440px. Формат JPG или WEBP, вес желательно до ~150 КБ.

## Что можно настроить

- Тексты, цены, состав тарифов, FAQ — прямо в `index.html`.
- Подписи к персонажам в галерее — в секции `<!-- GALLERY -->`.
- Скорость ленты галереи — `--52s` в правиле `.gmq-track` (`animation: gscroll 52s …`).
- Отключить анимации для слабых устройств уже учтено: при системной настройке «уменьшить движение» анимации выключаются автоматически.
